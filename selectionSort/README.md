## The Problem
Sort an array using Selection Sort

## Example
- The array: **[97, 5, 10, 0, -30]**
- Select Min: [97, 5, 10, 0 ```-30```]
- Swap: [```-30```, 5, 10, 0, ```97```]
- Select Min: [**-30,** 5, 10, ```0```, 97]
- Swap: [**-30,** ```0```, 10, ```5```, 97]
- Select Min: [**-30, 0,** 10, ```5```, 97]
- Swap: [**-30, 0,** ```5```, ```10```, 97]
- Select Min: [**-30, 0, 5,** ```10```, 97]
- Swap: [**-30, 0, 5,** ```10```, 97]
- Select Min: [**-30, 0, 5, 10,** ```97```]
- Swap: [**-30, 0, 5, 10,** ```97```]
- Sorted: **[-30, 0, 5, 10, 97]**
